# Nestor

AI-powered networking assistant for Palo Alto firewall and Panorama monitoring (Cloudflare Workers + React Router v7).

## Quick start

```bash
bun install
bun run dev
```

See [`AGENTS.md`](AGENTS.md) for stack details, 1Password Connect / secrets, and operations.

## Scripts

| Command | Purpose |
|--------|---------|
| `bun run dev` | Local dev server |
| `bun run build` | Production build |
| `bun run deploy` | Build and deploy to Cloudflare |
| `bun run typecheck` | TypeScript |
| `bun test` | Run tests (Bun) |
